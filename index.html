This is a jQuery plugin that can start some kind of animation
when entering the viewport
version 0.8 beta

written by Kiyoshi Honda

download

How to use

1.load 'risingSun.js' after jQury and jQuery easing.(This plugIn needs 'jQuery easing.')
2.The basic usage is $('element').RisingSun([otion]);
3.You can set some parameter to [option]. Check the reference.

Please bear in mind that scale of animation must be 'position:absolute';

Sample Animation

$("#fadein").RisingSun({animetime:300});
$("#scalein").RisingSun({animetime:500,scale:0.1,ease:"easeInCubic"});
$("#movein").RisingSun({animetime:800,x:50,y50,ease:"easeOutCubic"});
$("#popin").RisingSun({
     animetime:300,scale:0.1,popanime:true,ease:"easeOutCubic",replay:true
});
$("#wipein").RisingSun({
     nimetime:1200,wipe:"LR",ease:"easeInOutCubic",replay:true
});

Timer and Callback

$("#timer").RisingSun({
     timer:1500,animetime:300,scale:0.1,popanime:true,startPosition:0.7,ease:"easeOutCubic"
     onComplete:function(){
          $("#callback1").RisingSun({timer:0,animetime:300,scale:0.1,popanime:true,ease:"easeOutCubic"});
          $("#callback2").RisingSun({timer:100,animetime:300,scale:0.1,popanime:true,ease:"easeOutCubic"});
          $("#callback3").RisingSun({timer:200,animetime:300,scale:0.1,popanime:true,ease:"easeOutCubic"});
     }
});

Options

animetime: is animation speed [Default 500]
timer: is set Timer.Animation will start after this value(ms) [Default false]
startPosition: is startPosition at Viewport(top 0.0- bottom 1.0) [Default 0.5]
x: is x move Animation [Default 0]
y: is y move Animation [Default 0]
scale: is scale move Animation　[Default 1.0]
ease: takes an ease method　[Default 'linear']
wipe: is wipe animation("TB" or "LR") [Default false]
fadeStart: start opacity [Default 0]
fadeEnd: end opacity [Default 1.0]
popanime: Pop Up Animation　[Default false]
replay: replay [Default true]
onWakeup:function(){} is called when animation is sarted
onComplete:function(){} is called when animation is finished
onReplay:function(){} is called when replay
